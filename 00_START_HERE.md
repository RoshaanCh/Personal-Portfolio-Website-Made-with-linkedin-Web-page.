## ✏️ Quick Edit: Change Your Name

### Method 1: Simple Find & Replace
1. Right-click `index.html`
2. Select "Open with" → Notepad
3. Press Ctrl+H (Find & Replace)
4. Find: `Muhammad Roshaan`
5. Replace: `Your Name`
6. Click "Replace All"
7. Save (Ctrl+S)
8. Refresh browser to see changes

### Method 2: Open in Code Editor
1. Right-click `index.html`
2. Select "Open with" → Visual Studio Code
3. Use Ctrl+F to find text
4. Edit directly
5. Save file (Ctrl+S)
6. Website updates automatically!

## 🌐 Deploy Online (5 Minutes)

### Easiest Option: Netlify
1. Go to netlify.com
2. Click "Deploy manually"
3. Drag and drop `website` folder
4. Done! Your site is live! 🎉

### Alternative: GitHub Pages
1. Create GitHub account
2. Create new repository named `portfolio`
3. Upload website files
4. Enable Pages in settings
5. Your site is live at: `https://yourusername.github.io/portfolio`

**See DEPLOYMENT.md for detailed instructions**
